# Money Management App 💰

This is a simple money management web app I built using **HTML, CSS, and JavaScript** to practice DOM manipulation and basic frontend logic.

The main idea of this project is to track income and expenses, and see the balance update in real time.

---

## What this project does

* Add income and expense transactions
* Edit and delete transactions
* Automatically calculate total income, expense, and net balance
* Filter transactions using date range
* Sort transactions (newest / oldest)
* Form validation for correct inputs
* Dynamic sub-category selection based on income/expense

---

## Why I made this

I wanted to build something practical while learning JavaScript.
This project helped me understand:

* How to work with forms and user input
* DOM manipulation and event handling
* Managing data using arrays and objects
* Updating UI dynamically

---

## Tech Used

* HTML
* CSS
* JavaScript (no frameworks)

---

## How to run

Just open `index.html` in your browser.

---

## Project structure

* `index.html` – main UI
* `style.css` – styling
* `buttons.js` – core logic (add/edit/delete, calculations)
* `form.js` – category & subcategory handling
* `formvalidation.js` – input validation
